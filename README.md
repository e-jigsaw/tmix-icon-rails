# tmix-icon-rails

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tmix-icon-rails'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tmix-icon-rails

## Usage

```
@import "tmix-icon"
```
